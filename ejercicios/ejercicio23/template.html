Ejercicio 23

Crea una directiva ejercicio-tarjetas que tenga un atributo equipo que recibirá un array de elementos y otro atributo nombreEquipo

La directiva actuará de forma parecida a un ng-repeat.

Primero pintará en un h3 el nombre del equipo

Y a continuación repetirá el contenido transcluido tantas veces como elementos del array recibido, poniendo cada elemento en el scope transcluido con el nombre $heroe, y el índice de iteracion con $index

Resultado

El miembro nº {{$index}} del {{vm.nombreEquipo}} es {{$heroe.nombre}}

Código ejecutado

  <ejercicio-tarjetas nombreEquipo="vm.nombreEquipo" equipo="vm.equipo">
    <p>El miembro nº {{$index}} del {{vm.nombreEquipo}} es {{$heroe.nombre}}</p>
  </ejercicio-tarjetas>
